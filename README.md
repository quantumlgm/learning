<section>
  <h2 align="center">🤖 How I learn with AI. My Methodology</h2>
  <ol>
    <li>
      In the learning process, I use artificial intelligence as a mentor. For each new topic, I first ask the AI to generate a technical task that closely resembles real-world development problems.
    </li>
    <br />
    <li>
      Next, I independently analyze the task, design the architecture, and think about how to translate business logic into code. After that, I implement the solution, write the code, and perform initial testing.
    </li>
    <br />
    <li>
      Then I submit my implementation for AI code review, where I receive feedback, improvement suggestions, and error corrections. Based on this, I refine my code, learning from my mistakes and improving its structure and quality.
    </li>
    <br />
    <li>
      As a final step, I ask the AI to help me formulate a description of the completed work. I write English comments explaining the logic and design decisions in detail, which also helps me improve my technical English.
    </li>
    <br />
  </ol>
    
<p> This learning cycle helps me develop a deeper understanding of software engineering principles, especially in writing production-like, real-world-oriented code. </p>
    
</section>
